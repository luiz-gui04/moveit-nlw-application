# moveit-nlw-application
aplicação de pomodoro que usa um sistema de ranking e nivel para estimular o uso da técnica pomodoro, a cada ciclo de 25 minutos, a aplicação lhe oferece um exercício fisico como desafio, se completo, você receberá uma certa quantia de xp, e conforme mais xp for ganhando irá subir de nivel.

link: https://moveit-beta-three.vercel.app/
